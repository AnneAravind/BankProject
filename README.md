# Bank Project

A web-based Bank Management Application developed using Java Servlets, JDBC, and MySQL. The application allows users to manage bank account details through a simple interface and demonstrates core Java web-development concepts.

## Features
- Create new bank accounts
- View account details
- Update customer and account information
- Delete account records
- Perform CRUD operations
- Manage user sessions
- Store account data securely in a MySQL database

## Technologies Used
- Java
- Java Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## Concepts Used
- MVC Architecture
- Session Management
- CRUD Operations
- Database Connectivity
- Exception Handling

## How to Run
1. Clone this repository.
2. Create the required MySQL database and tables.
3. Update JDBC database connection details in the project.
4. Deploy the application on Apache Tomcat.
5. Open the application in a browser.
